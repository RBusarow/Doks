DocusyncSourceSet

abstract class DocusyncSourceSet @Inject constructor(val name: String, val objects: ObjectFactory) : RuleFactory, Serializable(source)

An abstract source set for docusync documentation and sample code. Provides a RuleFactory to allow for the creation of rules to be applied to the source set's documentation.
